# Azure with Keycloak

## Getting started

- Clone the repository
- Execute the ```deploy.sh``` script 

```bash
git clone
cd azuread-keycloak-oidc
./deploy.sh
```

This will create a resource group _Keycloak_ with all the resources into it.
